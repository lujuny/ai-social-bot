\#  Module 2: 趋势感知 (Competitor Scouting) 

**核心目标：** 自动去“发现”竞品的最新爆款，为后续分析提供素材。 
#### 1. 爆款抓取 
- **输入：** 读取 Module 1 的 `competitor_urls`。 

- **动作：** - 使用 `DrissionPage` (无痕模式) 打开竞品主页。  

  - 抓取**第一条置顶**或**最近一条点赞最高**的笔记。 

- **输出：** 提取关键信息：  

  - `reference_url`: 笔记详情页链接 (用于 Module 5 分析)。 

  - `title`: 笔记标题 (用于 Module 3 参考)。
#### 2. 简单策略路由 
- **逻辑：** 将抓取到的 `reference_url` 直接传递给 Module 5，启动分析流程。 
#### 3. 验收标准 
- [ ] 运行脚本，浏览器能自动打开小红书竞品主页。 
- [ ] 控制台能打印出竞品置顶笔记的标题和链接。
