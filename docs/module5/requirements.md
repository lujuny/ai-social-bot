# Module 5: 竞品归因分析 (Competitor Analysis)

**核心目标：** 分析 Module 2 抓到的那个“别人家的爆款”为什么火。

#### 1. 深度抓取
- **输入：** `reference_url` (来自 Module 2)。
- **动作：** - `DrissionPage` 进入详情页。
  - 抓取 **互动数据** (点赞/收藏/评论数)。
  - 抓取 **前 10 条热评** (这是用户反馈的精华)。

#### 2. AI 归因
- **动作：** 将“标题 + 正文 + 热评”喂给 LLM。
- **Prompt：** "这是一篇爆款，评论区都在夸什么？是夸干货多？还是夸图好看？请总结一个核心成功要素 (Winning Factor)。"
- **输出：** - `insight`: "用户非常喜欢 Step-by-Step 的教程格式。"

#### 3. 验收标准
- [ ] 能抓取到指定笔记的评论内容。
- [ ] AI 能输出一句具体的分析结论。
