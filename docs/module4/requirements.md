# Module 4: 自动化发布 (Auto Publishing)

**核心目标：** 将 Module 3 生成的内容真实填入发布框。

#### 1. 自动填表
- **输入：** Module 3 生成的内容包（图片路径、标题、正文）。
- **动作：** - 使用 `DrissionPage` (Auth Mode) 打开小红书创作中心。
  - **上传图片：** 定位 `<input type="file">` 并上传图片。
  - **填空：** 填入标题和正文。
  - **发布/存草稿：** 点击“发布”按钮（开发调试阶段建议改为点击“存草稿”或仅打印日志）。

#### 3. 验收标准
- [ ] 浏览器能自动完成上传图片和填写文字的动作。
